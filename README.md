# Trinity Cardápio

O Trinity Cardápio é um aplicativo de delivery de comida que visa facilitar a vida dos clientes ao permitir que eles façam pedidos online de forma rápida e segura.

## Funcionalidades

* Visualização do cardápio do restaurante
* Adição e remoção de itens do carrinho
* Calculo do total do pedido
* Opções de pagamento (Dinheiro e Pix)
* Envio de informações de pagamento para o WhatsApp
* Verificação da disponibilidade do restaurante

## Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Tailwind CSS para estilização
* Toastify para notificações

## Instalação

Para instalar as dependências do projeto, execute o comando `npm install` no diretório do projeto.

## Execução

Para executar o aplicativo, abra o arquivo `index.html` em um navegador web.
